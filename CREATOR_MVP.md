# 创作者助手 MVP（个人博主版）

## 1. 产品定位

这个模块不是“小红书运营中台”，而是你个人每天创作时用的“执行助手”：

- 帮你选今天写什么
- 帮你检查这篇能不能发
- 帮你输出可直接粘贴的发布包
- 帮你做最小复盘，形成自己的经验库

## 2. 功能清单（按优先级）

### P0（已实现）

1. 今日选题建议
- 入口：`改写工坊 -> 创作者助手 -> 今日选题建议`
- 按钮文案：`刷新选题`、`用作标题`、`填入素材区`
- 价值：降低“今天发什么”的决策成本

2. 发布前检查
- 入口：`改写工坊 -> 创作者助手 -> 发布前检查`
- 按钮文案：`一键检查`
- 检查项：标题长度、正文长度、段落数、标签数量、个人经历表达、互动引导
- 输出：`0-100` 分

3. 一键发布包
- 入口：`改写工坊 -> 创作者助手 -> 一键发布包`
- 按钮文案：`生成`、`复制`
- 输出内容：标题、正文、标签、首评建议（可直接粘贴）

4. 轻复盘
- 入口：`改写工坊 -> 创作者助手 -> 轻复盘`
- 按钮文案：`保存复盘`
- 字段：主题、链接、曝光、点赞、收藏、评论、涨粉、一句话复盘
- 输出：自动计算互动率，最近 5 条复盘记录

### P1（建议下一步）

1. 发布日历（周视图）
2. 选题去重（最近 14 天标题相似度提醒）
3. 复盘趋势（按周统计互动率变化）

## 3. 页面结构（当前）

页面：`改写工坊（Rewrite）`

1. 上半区：原有改写流程（素材输入、改写输出、导出草稿）
2. 下半区：创作者助手 4 卡片
- 今日选题建议
- 发布前检查
- 一键发布包
- 轻复盘

## 4. API 结构（当前）

新增接口：

- `GET /api/stats`
- `GET /api/creator/topics?limit=3`
- `GET /api/creator/reviews?limit=5`
- `POST /api/creator/reviews`

数据文件：

- 复盘数据：`data/creator_reviews.json`

## 5. 使用流程（每天）

1. 先点 `刷新选题`，选一个方向
2. 写完后点 `一键检查`
3. 通过后点 `生成` + `复制`，直接去发布
4. 发完回来填 `轻复盘`（30 秒完成）
